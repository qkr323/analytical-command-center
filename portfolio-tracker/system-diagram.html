🔄 Key Data Flows

📊 Position Sync Flow

Dashboard Sync Button → Click "IBKR" or "Binance"
Frontend → POST /api/proxy/sync/{broker}
Backend API → Fetch broker data (Flex API, REST API, or OpenD)
Database → Upsert positions, transactions, P&L calculations
Frontend → Display updated portfolio

📧 Email Sync Flow (Futu/SoFi)

Gmail → Daily statement PDF arrives from Futu/SoFi
Mac Terminal → Run python sync_from_email.py
Script → Connect to Gmail, download PDF, POST to /upload/statement
Backend → Parse PDF, extract positions & trades
Database → Upsert positions, recalculate P&L
Gmail → Label email "portfolio-synced"

💹 Price Refresh Flow

Scheduler → Every 30 minutes (automatic on Railway)
Backend → POST /sync/prices
Price APIs → yfinance (stocks), CoinGecko (crypto), frankfurter.app (FX)
Database → Update position prices & HKD values

🤖 AI Portfolio Analysis Flow

Dashboard → Click "Analyze Now" button
Frontend → POST /api/ai-analysis with portfolio data
Backend → Call Tavily API for recent market news
Backend → Call Groq LLM with portfolio + market context
Frontend → Display assessment, risks, trade ideas

🚀 Quick Start

Backend Setup

cd backend
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python -m uvicorn main:app --reload --port 8000

Frontend Setup

cd frontend
npm install
npm run dev
